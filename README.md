# School Management System (Backend - Phase 1)

Цей проєкт розробляється як частина навчального завдання. 
Розробники: Анастасія, Ангеліна і Настя .

## 🏗️ Архітектура моделей
У проєкті реалізовано такі сутності:
- **Branch**: Філії навчального центру.
- **Subject**: Предмети, прив'язані до філій.
- **Student**: База студентів.
- **StudentGroup**: Навчальні групи (зв'язок студентів, вчителів та предметів).
- **SubscriptionPlans**: Абонементи та контроль залишку занять.
- **Attendance**: Система відмітки відвідуваності.

## 🚀 Як запустити проєкт
1. Створити та активувати venv: `source venv/bin/activate`
2. Встановити залежності: `pip install django`
3. Виконати міграції: `python manage.py migrate`
4. Запустити сервер: `python manage.py runserver`
